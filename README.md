## 🌿 Sobre a EcoVolt

A **EcoVolt** é uma solução de ponta voltada para infraestrutura energética de grandes eventos. Nosso foco é transformar a forma como empresas gerenciam o consumo de energia, promovendo a transição para uma **Sociedade Carbono Zero** sem abdicar da eficiência operacional e rentabilidade.

Utilizamos inteligência de dados aplicada ao grid energético para reduzir custos operacionais em até **30%** e eliminar riscos regulatórios através de monitoramento preciso e em tempo real.

## 🚀 Tecnologias Utilizadas

Este projeto utiliza o que há de mais moderno no ecossistema Web para garantir escalabilidade e experiência premium:

- **Frontend**: [Next.js 15](https://nextjs.org/) (App Router, Server Components)
- **Estilização**: [Tailwind CSS 4](https://tailwindcss.com/)
- **Animações**: [Framer Motion](https://www.framer.com/motion/)
- **Iconografia**: [Lucide React](https://lucide.dev/)
- **Validação & Forms**: React Hook Form + [Zod](https://zod.dev/)
- **Temas**: Next Themes (Suporte nativo a Light & Dark Mode)
- **Linguagem**: [TypeScript](https://www.typescriptlang.org/)

## 📁 Estrutura do Repositório

O repositório está organizado de forma a separar a aplicação principal de eventuais outros serviços:

```bash
EcoVolt/
├── Leand-Page/         # Aplicação Next.js (Landing Page & Dashboard)
│   ├── public/         # Ativos estáticos (imagens, ícones)
│   └── src/            # Código-fonte da aplicação
│       ├── app/        # Rotas e layouts (Next.js App Router)
```

## 🛠️ Como Iniciar Localmente

### Pré-requisitos
- **Node.js**: v20 ou superior.
- **Gerenciador de pacotes**: NPM (ou similares como Yarn/Pnpm).

### Passo a Passo

1. **Clonar o repositório:**
   ```bash
   git clone https://github.com/HenriqueMC17/EcoVolt.git
   cd EcoVolt
   ```

2. **Navegar para a aplicação e instalar dependências:**
   ```bash
   cd Leand-Page
   npm install
   ```

3. **Rodar em modo desenvolvimento:**
   ```bash
   npm run dev
   ```
   Acesse: `http://localhost:3000`

## 🎨 Princípios de Design (Aesthetics & Performance)

O design da **EcoVolt** foi concebido para transmitir tecnologia, segurança e sustentabilidade:

1.  **Glassmorphism**: Transparências suaves e borrão de fundo para criar profundidade e modernidade.
2.  **Micro-interações**: Feedbacks visuais contínuos que guiam o usuário através das métricas.
3.  **Acessibilidade**: Seguindo rigorosamente os padrões WCAG para inclusão digital.
4.  **Cinematic Experience**: Uma narrativa visual que conta a história da eficiência energética.
